A 2D UFO racing game built with HTML5 Canvas and JavaScript, featuring a global leaderboard powered by Supabase.

## Features
- 2-player local multiplayer
- Solo mode with AI
- 4 difficulty levels
- 21 customizable UFO skins
- 8 background themes
- Shop system with in-game currency
- **Global leaderboard** (Supabase backend)

## Tech Stack
- HTML5 Canvas
- Vanilla JavaScript
- Supabase (Backend)
- Polish language UI

## Local Development
Simply open `index.html` in a modern web browser.
